Weather Challenge!

The goal of this website is to showcase tasks completed in an API challenge as part of the data analytics bootcamp I completed at Northwestern University. In this activity, I created a Python script to visualize the weather of 500+ cities across the world of varying distances from the equator. To accomplish this, I utilized the Python library https://pypi.python.org/pypi/citipy and the OpenWeatherMap API https://openweathermap.org/api.

On this site you have the ability look at data by weather type and by hemisphere by navigating the Plots dropdown menu.

I hope you enjoy the available information!
